# MOODCAST 情绪天气站架构

这个项目从私人 DJ 电台改成情绪天气站，保留“前端 App + Node 服务 + 外部天气 API”的结构。

## 1. 情绪天气界面

位置：`index.html`、`styles.css`、`app.js`

职责：

- 输入今日状态
- 调节身体能量和压力强度
- 展示情绪天气、能量值、行动建议
- 展示中英文推荐歌曲名
- 调用后端生成 moodcast

## 2. Node 服务

位置：`server.js`

职责：

- 获取时间和天气上下文
- 根据输入文本、能量、压力生成情绪天气
- 输出建议行动和歌曲推荐
- 保护未来的 API key，不让密钥暴露在浏览器里

当前接口：

- `GET /api/health`
- `GET /api/now`
- `GET /api/weather`
- `POST /api/moodcast`

## 3. 天气 API

当前使用 Open-Meteo：

- 不需要 API key
- 支持城市天气
- 支持浏览器经纬度定位天气

## 4. 歌曲推荐

当前只推荐歌名和歌手，不播放、不外链，避开音乐版权问题。

输出格式：

```json
{
  "title": "红豆",
  "artist": "王菲",
  "lang": "中文",
  "reason": "柔软但不沉重"
}
```

## 5. 后续 AI 接入

未来可以接 OpenAI，把当前规则推荐升级成更自然的 AI 输出。

输入：

- 今日状态文字
- 当前时间
- 天气
- 能量和压力

输出：

- 情绪天气
- 能量解释
- 今日建议
- 推荐歌曲名
- 可分享卡片文案
